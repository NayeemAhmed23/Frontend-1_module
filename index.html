Block Element

It is an element which starts from the new line and it also takes the complete width of the screen.

    Examples:

  1. h1-h6
  2. p
  3. div
  4. section

Inline Element

It is an element in which starts from the next point where the previous element ended.

    Examples:

  • img
  • vedio
  • audio
  • span
